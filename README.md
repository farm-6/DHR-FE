# HEC-FE 🦆
[Han River Ecology Collect] Front-End

### 🦆한강에오리 AI Front-End

> 서울의 공원 (https://parks.seoul.go.kr/) 에서 제공하는 한강생태지도의 한강 생태를 기준으로 도감을 구성하였습니다. </br>
![표지](https://github.com/farm-6/HEC-FE/assets/119389695/47fbe377-dc31-486a-8dfc-c4c70b71ecdf)


## 📌 기능 구성

1. 자랑하기
2. AI로 생태 분석
3. 도감
4. 마이페이지
